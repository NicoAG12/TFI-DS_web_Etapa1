# Fuentes revisadas

Esta carpeta guarda los archivos base usados para construir la Fase 4.

## Archivos

- `documento_base_fase4_ia.docx`: documento recibido con una version previa generada por IA.

## Uso

La nueva Fase 4 no copia literalmente la version previa. La usa como referencia inicial y la reescribe segun:

- El enunciado del TFI.
- La arquitectura real trabajada en Fase 1, Fase 2 y Fase 3.
- El contexto del sistema de canchas, reservas, pagos y matchmaking.
