# Terraform - Fase 5 IA

Esta carpeta muestra que recursos cloud se agregarian sobre la infraestructura de Fase 4/4B.

La idea no es duplicar EKS, RDS, Redis ni RabbitMQ. La Fase 5 solo incorpora recursos propios de IA:

- base vectorial administrada o endpoint externo;
- secretos para API keys;
- alarmas y metricas especificas de IA.
